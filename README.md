# Student Performance Dashboard (Streamlit)

## Run locally

From this folder:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Notes

- The app expects `Students_Grading_Dataset.csv` to be in the same folder as `streamlit_app.py` (default), but you can paste a different CSV path in the sidebar.
